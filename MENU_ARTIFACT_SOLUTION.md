# Решение проблемы с артефактом в меню

## Описание проблемы
При наведении на пункты основного меню в header.html появлялся артефакт - небольшой сегмент с округленными краями под выпадающим подменю.

## Причина проблемы
Проблема была вызвана свойством `border-radius: 8px;` в CSS-стиле для подменю:

```css
.header .submenu {
  /* ... другие свойства ... */
  border-radius: 8px;  /* Это создавало скругленные углы */
  /* ... */
}
```

## Решение
Были изменены следующие CSS-правила:

1. В основном стиле подменю (строка 453 в файле /workspace/styles/main.css):
   - `border-radius: 8px;` изменено на `border-radius: 0;`

2. В дублирующем стиле для десктопной версии (строка 251 в файле /workspace/styles/main.css):
   - Добавлено `border-radius: 0;`

## Результат
Теперь при наведении на пункты основного меню подменю появляется без скругленных краев, что устраняет визуальный артефакт.
