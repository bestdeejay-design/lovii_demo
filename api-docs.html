Документация по API

Техническая документация для интеграции с платформой Витрина

Введение

Наше API позволяет автоматически синхронизировать данные о товарах, остатках, заказах и ценах между вашей системой и платформой Витрина. Это упрощает управление магазином и снижает вероятность ошибок при обновлении информации.

Аутентификация

Для доступа к API используйте токен аутентификации, который можно получить в личном кабинете поставщика.

Authorization: Bearer YOUR_API_TOKEN

Доступные методы

  • Получение списка товаров
  • Обновление информации о товаре
  • Получение списка заказов
  • Обновление статуса заказа
  • Получение информации об остатках
  • Обновление цен

Частота запросов

Максимальная частота запросов: 1000 запросов в час на одного поставщика. Для высоконагруженных интеграций доступны специальные тарифы.

Примеры интеграции

Мы предоставляем SDK и примеры интеграции для популярных систем:

  • 1С:Предприятие 8
  • УТ 11
  • ERP системы
  • Собственные CRM системы
