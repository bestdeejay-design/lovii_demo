# Анализ проекта "Витрина"

## Общая структура проекта

Проект представляет собой веб-платформу "Витрина" для малого бизнеса, позволяющую самозанятым и ИП продавать свои товары через сервисы доставки. Структура проекта:

- HTML-страницы (19 файлов)
- CSS-файлы (2 файла)
- JavaScript-файлы (2 файла)
- Python-скрипт для рендеринга шаблонов
- Шаблонная система

## Текущие функциональные возможности

1. **Главная страница** - информация о проекте, аудитории и призывы к действию
2. **Каталог товаров** - фильтрация, сортировка, пагинация товаров
3. **Еда и доставка** - каталог заведений с фильтрами по городам и типам
4. **Регистрация/авторизация** - для покупателей и партнёров
5. **Темизация** - переключение между темами (тёмная/лесная)
6. **Адаптивный дизайн** - под мобильные устройства

## Обнаруженные проблемы

1. **Отсутствие страницы для отдельных заведений** - в отличие от товаров, у заведений нет уникальных страниц
2. **Статичные данные** - все товары и заведения хранятся в JavaScript как демонстрационные данные
3. **Отсутствие меню для заведений** - не реализована функциональность для просмотра меню заведений
4. **Нет информации о доставке в карточках заведений** - хотя есть фильтр по доставке, детали не отображаются
5. **Нет анимаций/эффектов** - карточки товаров и заведений не имеют интерактивных эффектов
6. **Неоптимизированные изображения** - используются изображения с Unsplash без локальной оптимизации
7. **Дублирование кода** - в файлах catalog/index.html и food-delivery/index.html много общего кода
