# Задание для разработчика: Исправление сайта "Витрина"

## Описание проблемы
Сайт "Витрина" не работает должным образом из-за неправильно сгенерированных HTML-страниц. Шаблонная система не была применена должным образом, что приводит к проблемам с отображением и функциональностью.

## Требуется выполнить

### 1. Проверить и исправить систему шаблонов
- Убедиться, что скрипт `render_templates.py` правильно генерирует все страницы
- Запустить скрипт для создания правильных HTML-файлов из шаблонов
- Проверить, чтобы все страницы использовали базовый шаблон `templates/base.html`

### 2. Проверить структуру файлов и навигацию
- Проверить, что все ссылки в навигации работают
- Убедиться, что все необходимые страницы существуют:
  - Главная (/)
  - Каталог (/catalog/)
  - Еда и доставка (/food-delivery/)
  - Для бизнеса (/business.html)
  - Для покупателей (/buyers.html)
  - Для партнёров (/partners.html)
  - Контакты (/contacts.html)
  - Условия (/terms.html)
  - Конфиденциальность (/privacy.html)
  - Помощь (/help.html)
  - Регистрация/вход для клиентов и партнёров

### 3. Проверить подкаталоги
- Убедиться, что подстраницы для разделов существуют:
  - /food-delivery/cafe/
  - /food-delivery/dark-kitchen/
  - /food-delivery/delivery/
  - /business/connect-company/
  - и т.д.

### 4. Проверить ресурсы
- Убедиться, что CSS файлы доступны по правильным путям
- Проверить, что JavaScript файлы загружаются корректно
- Убедиться, что все изображения и другие ресурсы доступны

### 5. Запустить генерацию недостающих страниц
- Выполнить скрипт `generate_pages.py` для создания недостающих подстраниц

### 6. Проверить функциональность
- Проверить, что переключение темы работает
- Убедиться, что мобильное меню открывается
- Проверить, что все кнопки и элементы управления работают
- Убедиться, что навигация по сайту работает корректно

## Приоритет
Высокий - сайт в настоящее время не работает должным образом
