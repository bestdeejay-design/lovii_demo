# Main Menu Project - Описание

## ВАЖНОЕ ПРАВИЛО ПРОЕКТА

При внесении любых изменений в любые файлы или страницы проекта:
1. Обязательно составлять план изменений и записывать его в соответствующий раздел файла pages.md ДО начала реализации
2. После реализации изменений - детально документировать все внесенные изменения в соответствующем разделе файла pages.md
3. Это правило является основой контроля качества и прозрачности разработки проекта
4. Все изменения должны быть задокументированы в pages.md или соответствующей странице для обеспечения целостности проекта

## Общее описание

Main Menu Project - это современное решение для создания адаптивного главного меню с подменю для веб-приложений. Проект разработан с использованием последних стандартов веб-разработки и обеспечивает отличный пользовательский опыт на всех устройствах.

## Архитектура проекта

Проект состоит из нескольких ключевых компонентов:

- HTML-шаблоны для структуры меню
- CSS-стили для визуального оформления и адаптивности
- JavaScript-функционал для интерактивности
- SCSS-файлы для легкой настройки и модификации

## Функциональные особенности

### Адаптивность
Меню автоматически адаптируется под размер экрана устройства. На мобильных устройствах используется гамбургер-меню, на десктопных устройствах - полноценное горизонтальное меню.

### Навигация по клику
В отличие от многих других решений, подменю открываются по клику, а не при наведении. Это улучшает пользовательский опыт, особенно на сенсорных устройствах.

### Поддержка подменю
Поддерживается многоуровневая структура меню с возможностью добавления вложенных подменю.

### Доступность
Меню разработано с учетом принципов доступности, что делает его удобным для пользователей с ограниченными возможностями.

## Технические детали

### HTML-структура
Меню построено на семантически правильной HTML-структуре с использованием тегов `nav`, `ul`, `li` и `button`.

### CSS-стилизация
Стили написаны с использованием современных возможностей CSS, включая Flexbox и Grid для раскладки, а также CSS-переменные для управления темой.

### JavaScript-функциональность
Интерактивность меню реализована с помощью чистого JavaScript без использования внешних библиотек. Обеспечивается плавное открытие/закрытие подменю, управление фокусом и клавиатурная навигация.

## Использование

Для интеграции меню в ваш проект:

1. Скопируйте необходимые файлы в ваш проект
2. Подключите CSS-файлы в секции head вашего HTML
3. Подключите JavaScript-файл перед закрывающим тегом body
4. Настройте структуру меню согласно вашим требованиям

## Кастомизация

Проект легко настраивается под ваши нужды:

- Цветовая схема может быть изменена через CSS-переменные
- Анимации и переходы могут быть модифицированы в CSS
- Структура меню может быть изменена в HTML
- Поведение меню может быть изменено в JavaScript
