# Страницы проекта «Лови»

## План реализации

### Основные страницы
- index.html - Главная страница
- about.html - О компании
- products.html - Продукты и услуги
- login.html - Страница входа
- profile.html - Личный кабинет
- contacts.html - Контакты

### Шаблоны
- /templates/header.html - Шапка сайта
- /templates/footer.html - Футер сайта
- /templates/mobile-nav.html - Мобильное меню

### Стили
- /css/main.css - Основные стили
- /css/themes.css - Переменные тем

### Скрипты
- /js/main.js - Основная логика
- /js/templates.js - Загрузка шаблонов
- /js/theme.js - Переключение тем

### Ресурсы
- /assets/fonts/ - Локальные шрифты
- /assets/icons/ - SVG иконки
