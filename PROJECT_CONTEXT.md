## О проекте

**LOVII** — SaaS-платформа локального маркетплейса.  
Объединяет малый бизнес (пекарни, цветы, химчистки, кофейни и т.д.) в единой цифровой инфраструктуре.

**Домен:** `lovii.mobiap.com` (GitHub Pages)

## Модель доходов (Revenue Model)

### Структура расходов с чека клиента

```
┌──────────────────────────────────────────────┐
│              ЧЕК КЛИЕНТА                      │
│              (100%)                           │
└────────────────────┬─────────────────────────┘
                     │
          ┌──────────┼──────────┐
          ▼          ▼          ▼
   ┌──────────┐ ┌────────┐ ┌──────────┐
   │ БАНК     │ │LOVII   │ │ ТОРГОВАЯ │
   │(эквайринг│ │комиссия│ │  ТОЧКА   │
   │ +вывод)  │ │  ~6%   │ │  ~90%    │
   │ 1.5–3.3% │ │        │ │          │
   └──────────┘ └────────┘ └──────────┘
```

### Банковские комиссии

| Операция | Ставка |
|----------|--------|
| Интернет-эквайринг (карта) | 1.7–2.5% |
| СБП (Система быстрых платежей) | от 0.8% |
| Вывод с номинального счёта партнёру/представителю | ~0.7% |
| **Итого банк (диапазон)** | **~1.5% – 3.3%** |

Процент динамический — зависит от выбора клиента (карта/SBP) и условий банка-партнёра.

### Доходы LOVII с торговой точки

| Тип дохода | Ставка | Условия |
|-----------|--------|--------|
| Комиссия с чека | **~6%** | Стандартная ставка |
| Абонентская плата | **2 990 ₽/мес** | Ежемесячно |
| Комиссия (акция) | снижена | Для точек с оборотом < 100 000 ₽/мес |

**Принцип:** общая нагрузка на точку (LOVII + банк) ≈ **~10%**.  
Больше брать нельзя — потеряем массовость и привлекательность для малого бизнеса.

### Сплит доходов LOVII (100% от комиссии 6% + абонентка)

```
        ┌──────────────────────────────────┐
        │     Доход с торговой точки        │
        │  (комиссия 6% + абонентка 2990)   │
        └────────────┬─────────────────────┘
                     │
           ┌─────────┼──────────┐
           ▼         ▼          ▼
     ┌──────────┐ ┌────────┐ ┌──────────┐
     │ 40%      │ │ 40%    │ │ 20%      │
     │ КОМПАНИЯ │ │ ПРЕД-  │ │ АМБАСАДОР│
     │(нужды    │ │ СТАВИ- │ │(обучение │
     │ компании)│ │ ТЕЛЬ   │ │мотивация)│
     └──────────┘ └────────┘ └──────────┘
```

- **40% Компания** — на развитие платформы, маркетинг, операции
- **40% Представитель** — за подключение и поддержку торговых точек на локации
- **20% Амбасадор** — за обучение и мотивацию представителей (половина от доли представителей из его сети)

### Статусы представителя

| Статус | Условия | Визуал |
|--------|---------|--------|
| **Цифровой Представитель** | До 30 торговых точек | 🥈 |
| **Цифровой Мэр** | От 30 торговых точек | 🥇 |
| **Цифровой Губернатор** | От 3 городов × 30 точек, выручка сети ≥ 15 млн ₽/мес | 👑 |

### Пример расчёта (1 торговая точка)

| Параметр | Значение |
|----------|---------|
| Средний чек | 1 200 ₽ |
| Заказов в день | 4 |
| Дней в месяц | 30 |
| **Выручка точки** | **144 000 ₽/мес** |
| Комиссия 6% | 8 640 ₽ |
| Абонентка | 2 990 ₽ |
| **Общий доход с точки** | **11 630 ₽/мес** |

**Сплит:**
- Компания (40%): 4 652 ₽
- Представитель (40%): 4 652 ₽
- Амбасадор (20%): 2 326 ₽

### Пример: Цифровой Губернатор

| Параметр | Значение |
|----------|---------|
| Городов | 3 |
| Точек на город | 30 |
| Всего точек | 90 |
| Средняя выручка точки | 144 000 ₽/мес |
| **Выручка всей сети** | **12 960 000 ₽/мес** |
| Комиссия 6% | 777 600 ₽ |
| Абонентка (90×2990) | 269 100 ₽ |
| **Общий доход сети** | **1 046 700 ₽/мес** |

**Сплит губернатора (40%):** 418 680 ₽/мес
