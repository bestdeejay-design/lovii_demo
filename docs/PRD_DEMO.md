> **Статус:** Сбор требований

## 1. КОНЦЕПЦИЯ

LOVII — платформа, объединяющая локальный бизнес (пекарни, цветы, химчистки, кофейни и т.д.) в едином маркетплейсе с приёмом заказов, программой лояльности и партнёрской сетью распространения.

### Экосистема

```
Компания → Амбасадор → Представитель → Партнёр (Точка) → Клиент
```

---

## 2. РОЛИ И ИХ ЭКРАНЫ

### 2.1. 👤 Клиент

Конечный покупатель. Заходит в маркетплейс, выбирает магазин, заказывает, получает баллы.

**Экраны:**
| Экран | Описание |
|-------|----------|
| **Главный** | Smart feed шорткатов, Flash Deals, подборки |
| **Каталог** | Сетка магазинов (2 колонки), пилюли-категории, поиск |
| **Магазин** | Баннер, товары, добавление в корзину |
| **Корзина** | Список, +/−, итого, оформить |
| **Профиль** | Баллы (крупно), QR-код, история заказов, избранное |
| **Баллы** | Баланс, история начислений/списаний |

---

### 2.2. 🏪 Партнёр (Торговая точка)

Магазин/бизнес, подключённый к платформе. Платит 6% с чека + 2 990 ₽/мес.

**Экраны:**
| Экран | Описание |
|-------|----------|
| **Дашборд** | Заказы сегодня, выручка, новые клиенты, быстрые действия |
| **Товары** | CRUD товаров, цены, активность, поиск |
| **Заказы** | Активные / История, статусы, действия |
| **Отчёты** | Графики выручки (день/неделя/месяц), детализация по чекам |
| **Лояльность** | Настройка кэшбэка, статистика баллов |
| **Настройки** | Данные точки, часы работы, профиль |

---

### 2.3. 🏙️ Представитель (локация)

Работает на локации, подключает торговые точки к платформе. Получает 40% дохода со своих точек.

**Статусы:**
| Статус | Условие | Доход представителя |
|--------|---------|-------------------|
| 🥈 **Цифровой Представитель** | До 30 точек | 40% дохода со своих точек |
| 🥇 **Цифровой Мэр** | От 30 точек | 40% + бейдж + привилегии |
| 👑 **Цифровой Губернатор** | 3+ города × 30+ точек, выручка сети ≥ 15 млн ₽/мес | 40% + все привилегии |

**Экраны:**
| Экран | Описание |
|-------|----------|
| **Дашборд** | Количество точек, доход, прогресс до следующего статуса |
| **Мои точки** | Список подключённых торговых точек с метриками |
| **Доход** | Детализация дохода (по дням/неделям/месяцам), история выплат |
| **Аналитика** | Графики роста сети, выручка точек |
| **Профиль** | Статус, бейдж, данные |

---

### 2.4. 🤝 Амбасадор

Получает коды от компании, выдает их представителям, обучает и мотивирует. Получает 20% дохода с точек сети своих представителей.

**Экраны:**
| Экран | Описание |
|-------|----------|
| **Дашборд** | Представители, точки в сети, доход, рейтинг |
| **Мои представители** | Список представителей, их точки, доход каждого |
| **Доход** | Детализация: доход с каждого представителя, история |
| **Обучение** | Материалы, инструкции для представителей |
| **Профиль** | Код амбасадора, рейтинг, данные |

## 3. СИСТЕМА РАСПРОСТРАНЕНИЯ (Distribution Model)

### 3.1. Цепочка подключения

```
┌──────────┐     код     ┌────────────┐    код     ┌──────────────┐
│ КОМПАНИЯ │ ──────────→ │ АМБАСАДОР  │ ──────────→ │ ПРЕДСТАВИТЕЛЬ│
│  LOVII   │             │            │             │              │
└──────────┘             └────────────┘             └──────┬───────┘
                                                            │
                                                   ┌────────▼────────┐
                                                   │ ТОРГОВАЯ ТОЧКА  │
                                                   │   (Партнёр)     │
                                                   └─────────────────┘
```

1. Компания выдаёт **специальный код** амбасадору
2. Амбасадор использует код для подключения **представителей**
3. Представитель работает на локации, подключает **торговые точки** (партнёров)
4. Торговая точка принимает заказы от **клиентов** и платит комиссию

### 3.2. Модель доходов

Доход с каждой торговой точки формируется из:
- **6% комиссия** с каждого заказа
- **2 990 ₽/мес** абонентская плата

### 3.3. Сплит доходов (100%)

```
      40% — КОМПАНИЯ        (развитие платформы, маркетинг, операции)
      40% — ПРЕДСТАВИТЕЛЬ   (за подключение и поддержку точек на локации)
      20% — АМБАСАДОР       (за обучение и мотивацию представителей)
```

#### Пример расчёта (1 точка)

| Параметр | Значение |
|----------|---------|
| Средний чек | 1 200 ₽ |
| Заказов/день | 4 |
| Дней/мес | 30 |
| Выручка точки | 144 000 ₽ |
| Комиссия 6% | 8 640 ₽ |
| Абонентка | 2 990 ₽ |
| **Итого доход с точки** | **11 630 ₽** |
| Компания (40%) | 4 652 ₽ |
| Представитель (40%) | 4 652 ₽ |
| Амбасадор (20%) | 2 326 ₽ |

#### Пример: сеть Цифрового Губернатора

| Параметр | Значение |
|----------|---------|
| Городов | 3 |
| Точек на город | 30 |
| Всего точек | 90 |
| **Выручка всей сети** | **~13 млн ₽/мес (≥ 15 млн с ростом)** |
| Доход представителя (40%) | ~418 680 ₽/мес |

## 6. СЦЕНАРИИ (User Flows) — предварительно

### Flow 1: Клиент покупает
```
Главная → Каталог → Выбор магазина → Добавление товара → 
Корзина → Оформление → Тост с номером → Баллы начислены
```

### Flow 2: Партнёр управляет точкой
```
Вход → Дашборд (заказы, выручка) → Новый заказ → 
Подтверждение → Статус «Готов» → Чек закрыт
```

### Flow 3: Представитель подключает точку
```
Дашборд → «+ Новая точка» → Форма (название, адрес, контакты) → 
Код активации → Точка в списке → Пошёл доход
```

### Flow 4: Амбасадор смотрит сеть
```
Дашборд → Список представителей → Выбор → Его точки → Доход
```

### Flow 5: Рост статуса представителя
```
Дашборд → Прогресс: 24/30 до «Мэр» → При подключении 30-й → 
Бейдж «Цифровой Мэр» 🥇 → Новые привилегии
```
