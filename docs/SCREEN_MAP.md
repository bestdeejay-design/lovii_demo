# Screen Map — LOVII Demo

> **Status:** Draft (awaiting approval)  
> **Navigation model:** Bottom Nav (4 tabs) + Role switch from Profile

---

## 1. Navigation Structure

```
┌──────────────────────────────────────────────────────────┐
│                      BOTTOM NAV                           │
│  [♥ Главная]  [🔍 Поиск]  [📦 Заказы]  [👤 Профиль]      │
└──────────────────────────────────────────────────────────┘
```

**Bottom Nav — 4 таба, всегда видимы.**  
Контент каждого таба может меняться в зависимости от выбранной роли.

---

## 2. Экран «Главная» (общий для всех)

Один экран для всех ролей.  
Показывает разный контент в зависимости от того, какая роль активна.

```
┌──────────────────────────────────┐
│  [Лого]                  [🔔][🦉] │
│  📍 Москва                       │
│                                  │
│  ┌─── Flash Deal ────────────┐   │
│  │ −20% на выпечку     [4 ч] │   │
│  └───────────────────────────┘   │
│                                  │
│  ┌─── Grid Menu (4×2) ──────┐   │
│  │ 🛍 📦 ⭐ 📊  |  👥 🏙 ⚡ ⚙ │   │
│  └───────────────────────────┘   │
│                                  │
│  ┌─── Context Block ─────────┐   │
│  │  (меняется по роли)        │   │
│  └───────────────────────────┘   │
│                                  │
│  [♥]  [🔍]  [📦]  [👤]          │
└──────────────────────────────────┘
```

**Context Block — что видят разные роли:**

| Роль | Context Block |
|------|--------------|
| 👤 **Клиент** | Активные заказы + баланс баллов |
| 🏪 **Партнёр** | Метрики (заказы сегодня, выручка) + новые заказы |
| 🏙️ **Представитель** | Сводка (точки, доход) + прогресс до следующего статуса |
| 🤝 **Амбасадор** | Сводка (представители, точки в сети, доход) |

> **Принцип:** экран один, блоки переключаются в зависимости от роли пользователя.

## 5. Экран «Профиль» + Выбор роли

**Ключевой экран для навигации по ролям.**

```
┌──────────────────────────────────┐
│  Профиль                         │
│                                  │
│  ┌──────────────────────────┐    │
│  │   👤 Александра           │    │
│  │   +7 (999) 123-45-67     │    │
│  │   [Изменить]             │    │
│  └──────────────────────────┘    │
│                                  │
│  Я —                            │
│  ┌────────┐ ┌────────┐         │
│  │ 👤     │ │ 🏪     │         │
│  │ Клиент │ │ Партнёр│         │
│  └────────┘ └────────┘         │
│  ┌────────┐ ┌────────┐         │
│  │ 🏙️     │ │ 🤝     │         │
│  │ Предст.│ │ Амбас. │         │
│  └────────┘ └────────┘         │
│                                  │
│  [🎯 Перейти в кабинет]          │
│                                  │
│  ───                              │
│  Мои данные                      │
│  → Бонусы и баллы                │
│  → Избранное                     │
│  → Настройки                     │
│  → Выйти                         │
└──────────────────────────────────┘
```

**Логика:**
1. На главной все видят общий экран
2. Хочешь управлять бизнесом → идёшь в Профиль
3. Выбираешь роль → нажимаешь «🎯 Перейти в кабинет»
4. Попадаешь на дашборд своей роли
5. Переключение между ролями — только через Профиль

## 6. Role Dashboards (отдельные экраны)

После выбора роли в Профиле → открывается дашборд этой роли.  
Каждый дашборд — свой Bottom Nav (другие иконки, не 4 общих).

### 6.1. 🏪 Партнёр (Торговая точка)

```
Bottom Nav: [📊 Дашборд] [📦 Товары] [📋 Заказы] [📈 Отчёты]
```

| Экран | Что содержит |
|-------|-------------|
| **Дашборд** | Метрики (заказы, выручка, новые клиенты) + последние заказы + быстрые действия (+Товар, Акция) |
| **Товары** | Список товаров с ценами, поиск, + добавить, активность (вкл/выкл) |
| **Заказы** | Входящие заказы (вкладки Активные / История), действия: Подтвердить → Готовится → Готов |
| **Отчёты** | График выручки (день/неделя/месяц), таблица чеков |

### 6.2. 🏙️ Представитель / 🥇 Мэр / 👑 Губернатор

```
Bottom Nav: [📊 Дашборд] [🏪 Точки] [💰 Доход] [👤 Профиль]
```

| Экран | Что содержит |
|-------|-------------|
| **Дашборд** | Статус (Представитель/Мэр/Губернатор) + прогресс-бар до след. статуса + ключевые метрики |
| **Точки** | Список подключённых точек с метриками (заказы, выручка) |
| **Доход** | График дохода, детализация по дням, история выплат |
| **Профиль** | Статус, бейдж, код представителя, данные |

**Мэр:** тот же набор + бейдж 🥇 + привилегии.  
**Губернатор:** + карта городов + сводка по всем городам.

### 6.3. 🤝 Амбасадор

```
Bottom Nav: [📊 Дашборд] [👥 Представители] [💰 Доход] [📚 Обучение]
```

| Экран | Что содержит |
|-------|-------------|
| **Дашборд** | Сводка (представители, точки, доход) + рейтинг/топ |
| **Представители** | Список представителей (имя, точки, доход) + детализация |
| **Доход** | Доход по представителям, история |
| **Обучение** | Материалы, инструкции |

---

## 8. Принципы навигации

1. **Главная — точка входа.** Всегда показывает актуальное для текущей роли
2. **Профиль — переключение ролей.** Только здесь можно сменить роль
3. **Роль сохраняется** в localStorage — при повторном визите та же роль
4. **По умолчанию — Клиент.** При первом входе роль = Клиент
5. **Bottom Nav меняется** когда в дашборде роли (но Главная, Поиск, Профиль доступны всегда)
6. **Кнопка «На главную»** в дашбордах выходит на общую Главную
