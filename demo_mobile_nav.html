Структура мобильной навигации

Этот пример демонстрирует структуру мобильной навигации, которая выезжает с правой стороны экрана. Она состоит из:

  • Кнопки-гамбургера (fixed position в правом нижнем углу)
  • Выезжающего меню с категориями и подкатегориями
  • Поддержки темной/светлой темы
  • Адаптивного дизайна для мобильных устройств

Для демонстрации навигация видна на всех размерах экрана, но в реальном проекте она отображается только на мобильных устройствах.
