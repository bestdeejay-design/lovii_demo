# Витрина — цифровая полка для малого бизнеса

## Описание проекта

"Витрина" — это платформа, которая помогает самозанятым и малым предприятиям продавать свои товары через популярные сервисы доставки. Мы устраняем барьеры между производителями и покупателями, делая местные продукты доступными в миллионах корзин.

## Архитектура проекта

Проект представляет собой многостраничный веб-сайт с использованием шаблонной системы. Основные компоненты архитектуры:

- **Шаблоны**: `/templates/base.html` — базовый шаблон для всех страниц
- **Контент**: Файлы `*_content.html` — содержимое для различных страниц
- **Генерация**: `render_templates.py` — скрипт для генерации страниц из шаблонов
- **Дополнительные страницы**: `generate_pages.py` — скрипт для генерации недостающих страниц

## Шаблоны и система рендеринга

### Базовый шаблон (`/templates/base.html`)
- Общий каркас для всех страниц
- Содержит общие элементы: шапку, навигацию, подвал
- Использует placeholder-ы: `{{ title }}`, `{{ content }}`, `{{ scripts }}`
- Поддерживает темную и светлую темы
- Включает SVG фон и мобильное меню

### Система генерации страниц
1. **Содержимое страниц** хранится в файлах `*_content.html`
2. **Шаблонизатор** заменяет переменные в `base.html` на реальный контент
3. **Результат** сохраняется в соответствующие HTML файлы

## Страницы сайта

### Основные страницы
1. **Главная страница** (`index.html`) — приветствует пользователей, объясняет концепцию платформы
2. **О нас** (`about.html`) — информация о проекте и его миссии
3. **Для бизнеса** (`business.html`) — информация для продавцов и производителей
4. **Для покупателей** (`buyers.html`) — информация для конечных пользователей
5. **Для партнёров** (`partners.html`) — информация для партнёров и интеграторов
6. **Каталог** (`/catalog/`) — список товаров от местных производителей
7. **Еда и доставка** (`/food-delivery/`) — информация о доставке еды
8. **Контакты** (`contacts.html`) — контактная информация
9. **Условия использования** (`terms.html`) — юридическая информация
10. **Политика конфиденциальности** (`privacy.html`) — политика обработки данных
11. **Помощь** (`help.html`) — часто задаваемые вопросы и инструкции

### Страницы авторизации
12. **Вход покупателя** (`login-customer.html`) — страница входа для покупателей
13. **Вход партнёра** (`login-partner.html`) — страница входа для партнёров
14. **Регистрация покупателя** (`register-customer.html`) — регистрация для покупателей
15. **Регистрация партнёра** (`register-partner.html`) — регистрация для партнёров

### Подразделы и категории
- **business/** — подразделы для бизнеса (connect-company, support, pro-tariffs, api)
- **buyers/** — подразделы для покупателей (how-to-buy, cashback, delivery-payment, faq)
- **partners/** — подразделы для партнёров (banks, integrators, media)
- **food-delivery/** — подразделы для доставки еды (cafe, dark-kitchen, delivery, establishments-sales)

## Навигационная структура

Сайт имеет многоуровневую навигацию:
- **Главное меню**: содержит основные разделы сайта
- **Выпадающие меню**: раскрывают подразделы для каждого основного раздела
- **Мобильная навигация**: реализована в виде off-canvas меню с FAB-кнопкой

## Технические особенности

- Адаптивный дизайн для всех устройств
- Поддержка темной и светлой темы (сохранение выбора в localStorage)
- Локализация на русском языке
- Использование современных CSS-фич (CSS-переменные, Grid, Flexbox)
- Чистый JavaScript без фреймворков
- Шаблонная система для генерации страниц
- Автоматизированный скрипт для создания новых страниц
- Мобильная навигация с FAB-кнопкой

## Установка и запуск

Проект не требует установки зависимостей. Просто откройте `index.html` в браузере или разместите на любом хостинге, поддерживающем статические сайты.

Для генерации новых страниц используйте:
```bash
python generate_pages.py
```

Для рендеринга шаблонов используйте:
```bash
python render_templates.py
```
