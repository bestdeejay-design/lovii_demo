# Витрина - Онлайн-платформа для торговли

## Обзор проекта

"Витрина" - это современная онлайн-платформа, которая позволяет малому бизнесу подключаться к тысячам покупателей через приложение, сайт и сеть точек выдачи. Проект представляет собой многостраничный веб-сайт с адаптивным дизайном, который обслуживает как покупателей, так и продавцов.

## Структура проекта

```
/workspace/
```

## Связи между файлами

1. **index.html** загружает шаблоны через JavaScript:
   - templates/header.html → элемент с id="header-container"
   - templates/footer.html → элемент с id="footer-container"
   - templates/mobile-nav.html → элемент с id="mobile-nav-container"

2. **CSS связь**:
   - Все HTML-файлы используют styles/main.css для оформления

3. **JavaScript связь**:
   - index.html использует js/main.js для функций:
     - Загрузка шаблонов
     - Инициализация мобильного меню
     - Переключение темы
     - Обработка форм

4. **Изображения**:
   - Все HTML-файлы ссылаются на изображения в директории images/

## Особенности реализации

1. **Шаблонизация**: Используется JavaScript для загрузки общих компонентов (шапка, футер, мобильное меню)

2. **Адаптивный дизайн**: Поддержка мобильных и десктопных устройств

3. **Темная тема**: Возможность переключения светлой/темной темы

4. **Модульность**: Компоненты разнесены по шаблонам для повторного использования
