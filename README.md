# Витрина - Онлайн-платформа для торговли

## Обзор проекта

"Витрина" - это современная онлайн-платформа, которая позволяет малому бизнесу подключаться к тысячам покупателей через приложение, сайт и сеть точек выдачи. Проект представляет собой многостраничный веб-сайт с адаптивным дизайном, который обслуживает как покупателей, так и продавцов.

## Структура проекта

## Описание страниц и их функциональность

### 1. index.html (Главная страница)
- **Назначение**: Основная страница сайта, представляющая проект "Витрина"
- **Функциональность**:
  - Герой-секция с призывом к действию
  - Раздел "Как это работает" с 3 шагами для начала продаж
  - Раздел "Для кого подходит" с категориями бизнеса
  - Форма для отправки заявки
  - Контактная информация
  - Загрузка шаблонов через JavaScript

### 2. templates/header.html (Шапка сайта)
- **Назначение**: Шаблон десктопного меню и логотипа
- **Функциональность**:
  - Логотип "Витрина" с ссылкой на главную
  - Меню "Для клиентов" с подменю
  - Меню "Для бизнеса" с подменю
  - Ссылки на разделы "О проекте" и "Контакты"
  - Переключатель темы
  - Кнопка мобильного меню

### 3. templates/mobile-nav.html (Мобильная навигация)
- **Назначение**: Шаблон нижней навигационной панели для мобильных устройств
- **Функциональность**:
  - Кнопка бургера для открытия меню
  - Логотип "Витрина" в шапке мобильного меню
  - Мобильное меню с теми же разделами, что и десктопное
  - Ссылки на вход и корзину
  - Переключатель темы

### 4. templates/footer.html (Футер сайта)
- **Назначение**: Шаблон нижнего колонтитула сайта
- **Функциональность**:
  - Информация о компании
  - Навигация по сайту
  - Социальные ссылки
  - Копирайт

## Связи между файлами

1. **index.html** загружает шаблоны через JavaScript:
   - templates/header.html → элемент с id="header-container"
   - templates/footer.html → элемент с id="footer-container"
   - templates/mobile-nav.html → элемент с id="mobile-nav-container"

2. **CSS связь**:
   - Все HTML-файлы используют styles/main.css для оформления

3. **JavaScript связь**:
   - index.html использует js/main.js для функций:
     - Загрузка шаблонов
     - Инициализация мобильного меню
     - Переключение темы
     - Обработка форм

4. **Изображения**:
   - Все HTML-файлы ссылаются на изображения в директории images/

## Особенности реализации

1. **Шаблонизация**: Используется JavaScript для загрузки общих компонентов (шапка, футер, мобильное меню)

2. **Адаптивный дизайн**: Поддержка мобильных и десктопных устройств

3. **Темная тема**: Возможность переключения светлой/темной темы

4. **Модульность**: Компоненты разнесены по шаблонам для повторного использования
