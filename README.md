# Витрина - Сайт доставки еды с кэшбэком

Современный, стильный дизайн сайта для платформы «Витрина» с акцентом на доставку еды и кэшбэк для пользователей.

## Особенности

- **Современный дизайн**: Чистый, минималистичный интерфейс с акцентом на аппетитные фотографии еды
- **Адаптивность**: Полностью адаптивный дизайн для мобильных и десктопных устройств
- **Финтех-бренд**: Использование цветовой схемы и элементов дизайна, характерных для финтех-приложений
- **Интуитивная навигация**: Простая и понятная навигация с выпадающими меню
- **Доступность**: Поддержка клавиатурной навигации и фокусных состояний

## Технические характеристики

- **Цвета**:
  - Основной фон: белый (#FFFFFF) или светло-кремовый (#FAFAF7)
  - Акцентные цвета: глубокий зелёный (#2A5C34) и светло-зелёный (#E8F5E9)
- **Шрифты**: Inter (без засечек)
- **Иконки**: Lucide Icons
- **Элементы**: Мягкие закругления (border-radius: 12px), тени, плавные переходы

## Структура страницы

### Desktop (≥769px):
- Шапка: логотип слева, навигация по центру
- Выпадающие меню при наведении
- Геро-секция с метриками и CTA-кнопками
- Карточки категорий еды
- Раздел с партнёрами
- Подвал с контактами и политиками

### Mobile (≤768px):
- Выезжающее меню справа
- Аккордеон для подменю
- Увеличенные отступы и размеры кнопок
- Оптимизированные размеры шрифтов

## Компоненты

- **Выпадающие меню**: Для разделов "Каталог" и "Еда и доставка"
- **Карточки блюд**: С тенью, закруглёнными углами, эффектом при наведении
- **Кнопки**: Округлые с плавными hover-эффектами
- **Метрики**: Показатели "ресторанов", "кэшбэка", "доставки"
- **Партнёры**: Логотипы банков и партнёров

## Тон общения

Лаконичный, деловой, но дружелюбный. Заголовки без использования "Вы":
- "Еда у вас дома за 20 минут"
- "Кэшбэк на каждую доставку"
- "Подключите свой ресторан — бесплатно"

## Файлы проекта

- `index.html` - основная HTML-структура
- `styles.css` - стили и дизайн
- `script.js` - интерактивные элементы
- `README.md` - текущий файл документации

## Запуск

Для просмотра сайта достаточно открыть файл `index.html` в любом современном браузере.

## Адаптивность

Сайт полностью адаптивен и не требует горизонтальной прокрутки на мобильных устройствах. Все элементы корректно отображаются на различных размерах экранов.

## Состояния интерактивных элементов

- Hover: изменение цвета, тени, позиции
- Focus: видимая рамка для доступности
- Active: кратковременные анимации при клике

## Использование

Проект готов к передаче разработчику или импорту в Figma/Webflow. Все элементы стилизованы и функциональны.
