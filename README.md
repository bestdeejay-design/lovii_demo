# Витрина — цифровая полка для малого бизнеса

## Описание проекта

"Витрина" — это платформа, которая помогает самозанятым и малым предприятиям продавать свои товары через популярные сервисы доставки. Мы устраняем барьеры между производителями и покупателями, делая местные продукты доступными в миллионах корзин.

## Страницы сайта

1. **Главная страница** (`index.html`) — приветствует пользователей, объясняет концепцию платформы
2. **Каталог** (`/catalog/`) — список товаров от местных производителей
3. **О нас** (`about.html`) — информация о проекте и его миссии
4. **Для бизнеса** (`business.html`) — информация для продавцов и производителей
5. **Контакты** (`contacts.html`) — контактная информация
6. **Корзина** (`cart.html`) — страница с товарами, выбранными пользователем
7. **Товар** (`product.html`) — страница с информацией о конкретном товаре
8. **Условия использования** (`terms.html`) — юридическая информация
9. **Политика конфиденциальности** (`privacy.html`) — политика обработки данных
10. **Помощь** (`help.html`) — часто задаваемые вопросы и инструкции

## Технические особенности

- Адаптивный дизайн для всех устройств
- Поддержка темной и светлой темы
- Локализация на русском языке
- Использование localStorage для хранения данных корзины
- Современные CSS-фичи (CSS-переменные, Grid, Flexbox)
- Чистый JavaScript без фреймворков

## Установка и запуск

Проект не требует установки зависимостей. Просто откройте `index.html` в браузере или разместите на любом хостинге, поддерживающем статические сайты.
