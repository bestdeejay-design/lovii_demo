Интеграция с API

Автоматизируйте бизнес-процессы через API и интеграцию с 1С

Документация REST API

Наше API позволяет автоматизировать все ключевые бизнес-процессы: загрузку товаров, управление остатками, обработку заказов и получение аналитики.

Возможности API:

  • Загрузка и обновление товаров
  • Синхронизация остатков
  • Получение и обработка заказов
  • Управление ценами
  • Получение аналитических данных
  • Работа с изображениями

Примеры кода

Python

import requests

# Авторизация
headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Получение списка товаров
response = requests.get('https://api.vitrina.ru/products', headers=headers)
products = response.json()

PHP

<?php
$curl = curl_init();

curl_setopt_array($curl, [
    CURLOPT_URL => "https://api.vitrina.ru/products",
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        "Authorization: Bearer YOUR_API_KEY",
        "Content-Type: application/json"
    ]
]);

$response = curl_exec($curl);
curl_close($curl);

$products = json_decode($response, true);

Тестовое окружение (sandbox)

Для тестирования интеграции мы предоставляем специальное тестовое окружение, в котором можно безопасно экспериментировать с API без риска повлиять на реальные данные.

Доступы к песочнице:

  • URL: https://sandbox.vitrina.ru/api
  • API Key: sandbox_123456789
  • Тестовые данные: предоставлены для отладки

Получить API-ключ

Для доступа к API необходимо получить персональный ключ в личном кабинете.
