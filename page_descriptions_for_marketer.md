# Описание страниц для маркетолога

Данный документ содержит описание всех страниц сайта "Витрина" с указанием их назначения, целевой аудитории и рекомендаций по наполнению контентом для маркетолога.

## Главная страница (index.html)

**Назначение:** Основная страница сайта, которая формирует первое впечатление о платформе. Главная цель - привлечь внимание, объяснить ценность платформы и побудить к действию (регистрация поставщика или покупка товаров).

**Целевая аудитория:** 
- Потенциальные поставщики (самозанятые, ИП, ООО)
- Покупатели
- Владельцы гипермаркетов и ПВЗ

**Ключевые элементы:**
- Герой-секция с основным сообщением: "Витрина — ваш товар на полке гипермаркета"
- Краткое описание преимуществ платформы
- Кнопки призыва к действию: "Смотреть каталог" и "Стать поставщиком"
- Раздел "Как начать продавать за 3 шага"
- Раздел "Для кого подходит Витрина"
- Форма для заявки на подключение
- Рекомендательные товары

**Рекомендации по контенту:**
- Использовать убедительные заголовки и подзаголовки
- Добавить социальные доказательства (статистика, отзывы)
- Подчеркнуть уникальное торговое предложение (USP)
- Использовать призывы к действию (CTA) в разных частях страницы

## Заключение для маркетолога

При наполнении контента на все эти страницы рекомендуется:

1. **Сохранять единый стиль и тон общения** - использовать дружелюбный, профессиональный тон
2. **Сосредоточиться на пользе для пользователя** - всегда отвечать на вопрос "Что это дает мне?"
3. **Использовать конкретные цифры и факты** - статистика, социальные доказательства
4. **Добавлять призывы к действию** - направлять пользователей к следующему шагу
5. **Оптимизировать для SEO** - использовать ключевые слова, правильные заголовки
6. **Создавать уникальный и ценный контент** - избегать копирования, добавлять экспертизу
7. **Регулярно обновлять информацию** - особенно цены, тарифы и контактную информацию
