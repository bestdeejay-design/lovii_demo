# Структура мобильной навигации выезжающей сбоку

## Общая концепция
Мобильная навигация представляет собой выезжающее меню, которое появляется с правой стороны экрана при клике на кнопку-гамбургер. Это популярное решение для мобильных интерфейсов, позволяющее экономить пространство на экране, но обеспечивать доступ ко всем основным разделам сайта.

## HTML-структура

### Основные элементы
```html
<div class="mobile-nav-wrapper">
    <!-- Кнопка для открытия меню -->
    <div class="mobile-nav-button" id="mobileNavButton">
        <div class="hamburger">
            <span></span>
            <span></span>
            <span></span>
        </div>
    </div>
    
    <!-- Основное меню -->
    <div class="mobile-nav-menu" id="mobileNavMenu">
        <!-- Шапка меню -->
        <div class="mobile-nav-header">
            <div class="logo"><a href="#home" id="mobileLogo">Логотип</a></div>
            <button class="close-btn" id="closeMobileMenu">×</button>
        </div>
        
        <!-- Список пунктов меню -->
        <ul class="mobile-nav-list">
            <!-- Пункт меню с подменю -->
            <li class="nav-item has-submenu">
                <a href="#" class="nav-link">Название раздела <i class="fas fa-chevron-down submenu-arrow"></i></a>
                <ul class="submenu">
                    <li><a href="#"><i class="fas fa-icon"></i> Подпункт</a></li>
                </ul>
            </li>
            
            <!-- Обычный пункт меню -->
            <li class="nav-item">
                <a href="#" class="nav-link"><i class="fas fa-icon"></i> Название</a>
            </li>
            
            <!-- Разделитель -->
            <li class="nav-divider"></li>
        </ul>
    </div>
</div>
```

## CSS-стили

### Основные свойства
- `.mobile-nav-wrapper` - контейнер, позиционируется фиксированно в правом нижнем углу
- `.mobile-nav-button` - круглая кнопка с гамбургером, содержит анимацию при наведении
- `.mobile-nav-menu` - само меню изначально скрыто за правым краем экрана (`right: -340px`) и появляется при активации (`right: 20px`)

### Анимация
- Плавное появление меню с помощью CSS transition
- Поворот стрелочки подменю при раскрытии
- Эффекты при наведении на элементы меню

## JavaScript-функциональность

### Основные действия
1. **Открытие/закрытие меню** - по клику на кнопку-гамбургер добавляется/удаляется класс `active`
2. **Закрытие по клику вне области** - проверяется, был ли клик вне меню
3. **Работа с подменю** - по клику на пункт с подменю раскрывается соответствующий список
4. **Переключение темы** - интеграция с системой темизации сайта
5. **Прокрутка к верху** - при клике на логотип

## Особенности реализации

### Адаптивность
- Меню отображается только на мобильных устройствах (через медиазапросы)
- Для демонстрации может быть принудительно включено

### Доступность
- Использование семантических тегов
- Поддержка клавиатурной навигации
- Четкая визуальная обратная связь при взаимодействии

### Производительность
- Минимальное количество DOM-элементов
- Использование CSS-анимаций вместо JavaScript где возможно
- Оптимизация рендеринга при прокрутке подменю

## Интеграция с основным проектом

Файл шаблона находится по пути: `/workspace/templates/mobile-nav.html`
Соответствующие стили находятся в: `/workspace/styles/main.css` (секция ".mobile-nav")
JavaScript-функциональность реализована в: `/workspace/js/main.js` (функция "initializeNewMobileNavigation")
