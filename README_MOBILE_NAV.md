# Мобильная навигация выезжающая сбоку

## Описание

В этом проекте реализована мобильная навигация, которая выезжает с правой стороны экрана. Это современное решение для мобильных интерфейсов, которое позволяет сохранить пространство на экране, но при этом обеспечивает легкий доступ ко всем основным разделам сайта.

## Файлы проекта

- `templates/mobile-nav.html` — шаблон мобильной навигации
- `styles/main.css` — стили для мобильной навигации (раздел .mobile-nav)
- `js/main.js` — JavaScript-функциональность (функция initializeNewMobileNavigation)
- `demo_mobile_nav.html` — демонстрационный файл с полной работоспособной навигацией
- `MOBILE_NAV_STRUCTURE.md` — подробное описание структуры навигации

## Особенности реализации

- **Позиционирование**: кнопка меню находится в правом нижнем углу экрана
- **Анимация**: плавное появление меню с правой стороны
- **Поддержка подменю**: возможность создания многоуровневых меню
- **Темизация**: поддержка светлой и темной темы
- **Адаптивность**: отображается только на мобильных устройствах
- **Доступность**: семантическая разметка и поддержка клавиатурной навигации

## Структура HTML

Мобильная навигация состоит из:
1. Кнопки-гамбургера (`.mobile-nav-button`)
2. Выезжающего меню (`.mobile-nav-menu`)
3. Заголовка меню с логотипом и кнопкой закрытия
4. Списка навигационных элементов (включая подменю)
5. Разделителей и специальных элементов (вход, корзина, переключение темы)

## Использование

Для запуска демонстрационного файла:
1. Убедитесь, что локальный сервер запущен: `python3 -m http.server 8000`
2. Откройте в браузере: http://localhost:8000/demo_mobile_nav.html
3. Нажмите на кнопку-гамбургер в правом нижнем углу экрана

## Адаптация под другие проекты

Для использования этой навигации в другом проекте:
1. Скопируйте HTML-разметку из файла шаблона
2. Добавьте соответствующие CSS-стили
3. Подключите JavaScript-функциональность
4. При необходимости адаптируйте пути к страницам и иконкам

## Совместимость

- Совместимо со всеми современными мобильными браузерами
- Работает на устройствах с сенсорным экраном
- Поддерживает как портретную, так и альбомную ориентацию
