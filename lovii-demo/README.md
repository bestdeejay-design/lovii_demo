# Сайт «Лови» - Техническая документация

## Описание проекта
Полностью автономный, кликабельный B2B-сайт для проекта «Лови» (SaaS + оператор лояльности), который:
- работает локально (без интернета),
- не использует внешние CDN (все ресурсы — в репозитории),
- поддерживает адаптивность и смену тем (тёмная — по умолчанию),
- построен на шаблонах (DRY-подход),
- соответствует UI/UX лучших мировых медиа (BBC, CNN, The Guardian, Amazon),
- легко поддерживается и расширяется без разрушения структуры.

## Структура проекта
```
/lovii-demo
├── index.html
├── about.html
├── products.html
├── login.html
├── profile.html
├── contacts.html
├── /templates
│   ├── header.html
│   ├── footer.html
│   └── mobile-nav.html
├── /css
│   ├── main.css          ← единый CSS-файл (все стили)
│   └── themes.css        ← только переменные тем
├── /js
│   ├── main.js           ← общая логика
│   ├── templates.js      ← загрузка шаблонов
│   └── theme.js          ← переключение тем
└── /assets
    ├── fonts/
    │   └── Inter-*.woff2 ← локальные шрифты
    └── icons/
        └── *.svg
```

## Особенности реализации
- **Локальные ресурсы**: Все шрифты и иконки подключены локально
- **Система шаблонов**: Шапка, футер и мобильное меню загружаются динамически
- **Темы**: Тёмная (по умолчанию) и светлая тема с сохранением в localStorage
- **Адаптивность**: Поддержка мобильных устройств, планшетов и десктопов
- **Мобильное меню**: Фиксированное меню внизу экрана для удобства на мобильных устройствах
- **UI/UX**: Реализованы принципы дизайна от ведущих мировых медиа

## Запуск проекта
Проект запускается локально через браузер (file:// протокол), без необходимости в веб-сервере.

## Требования к реализации
- Все изменения сначала фиксируются в pages.md
- Структура проекта не меняется без согласования
- Перед коммитом проверяется:
  - Работает ли переключение тем?
  - Корректно ли загружаются шаблоны?
  - Нет ли внешних CDN?
  - Проходит ли валидацию HTML/CSS?
- Все стили — только через css/main.css
- Все скрипты — только через /js/
