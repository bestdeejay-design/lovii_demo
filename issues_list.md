# СПИСОК ПРОБЛЕМ САЙТА "ВИТРИНА"

## Классификация проблем

### Критические (требуют немедленного исправления)
1. **Некоторые внутренние ссылки ведут на якоря (#) вместо реальных URL**
   - В мобильной навигации встречаются ссылки с якорями (#products, #food и т.д.)
   - Эти ссылки не обеспечивают навигацию между страницами в многостраничном приложении

### Серьёзные (требуют исправления в ближайшее время)
1. **Формы не имеют реальной функциональности отправки**
   - Формы имеют только клиентскую валидацию
   - Отправка данных на сервер не реализована (используются только алерты)
   
2. **Некоторые ссылки в мобильной навигации используют якоря вместо URL**
   - Вместо перехода на соответствующие странице, используются якорные ссылки
   
3. **Нет полноценной системы управления состоянием корзины**
   - Функциональность корзины ограничена прототипом без сохранения данных

4. **Некорректное количество страниц в отчете**
   - В отчете указано 46 страниц, но на самом деле найдено 47 HTML файлов (включая index-business.html)

### Незначительные (можно исправить в будущем)
1. **Нет специфичных метатегов для SEO на всех страницах**
   - Отсутствуют уникальные title и description для большинства страниц
   
2. **Некоторые изображения могут быть оптимизированы**
   - Нет использования WebP форматов или других методов оптимизации
   
3. **Нет систем ленивой загрузки контента**
   - Изображения и другие ресурсы загружаются сразу, без lazy loading

4. **Шаблоны в директории templates не учитываются как отдельные страницы**
   - Эти файлы используются как включения, но отмечены как страницы в карте сайта
