# Дизайн-система сайта LOVII (финальная версия)

## 1. Общие принципы
- Стиль: Современный, тёплый, цифровая экономика малого бизнеса
- Тон: Дружелюбный, доверительный, профессиональный — без маркетингового пафоса
- Аудитория: Микробизнес (СЗ, ИП), франчайзи, HR-отделы, партнёры (Mobiap, Яндекс, Igoods, Фотострана)
- Язык: Русский
- Адаптивность: Full responsive (mobile, tablet, desktop)

## 2. Цветовая палитра (точные HEX-коды)

### Основные брендовые цвета
- Primary (терракотовый): #C97C5D — кнопки, логотип, акценты
- Secondary (шалфейный): #8F9E8B — иконки, второстепенные элементы
- Background (охристый / тёплый бежевый): #E9E4D4 — фон сайта и карточек
- Surface (светлый фон блоков): #F0EDDE — фон карточек, модальных окон

### Цвета текста
- Text Primary (тёмно-зелёный): #4A5447 — основной текст, заголовки
- Text Secondary: #5F6B5C — подписи, описания

### Акцентные и дополнительные
- Accent / Soft Gold: #C9A66B — премиальные метки, официальные стикеры
- Muted Blush (нежный розовый): #E2B7A9 — для категорий "красота", "дети"

## 3. Типографика
- Основной шрифт: TT Norms Pro или Manrope (без засечек, тёплый, читаемый)
- Размеры:
  - H1: 48px (desktop) / 32px (mobile)
  - H2: 36px / 24px
  - H3: 24px / 20px
  - Body: 18px / 16px
  - Caption: 14px
- Межстрочный интервал: 1.5
- Цвет текста по умолчанию: #4A5447

## 4. Компоненты

### Кнопки
- Primary: фон #C97C5D, текст белый (#FFFFFF), скругление 12px, padding 12px 24px
- Secondary: border 2px solid #8F9E8B, фон прозрачный, текст #8F9E8B
- Accent (premium): фон #C9A66B, текст #4A5447

### Карточки
- Фон: #F0EDDE
- Тень: 0 4px 12px rgba(201, 124, 93, 0.1)
- Скругление: 16px
- Padding: 24px

### Иконки
- Формат: SVG
- Цвет по умолчанию: #8F9E8B
- Акцентные иконки: #C97C5D или #C9A66B
- Размер: 24x24px

## 5. Темы (опционально)
Сайт может поддерживать переключение:
1. **Тёплая (default)** — как выше
2. **Светлая контрастная** — фон #FFFFFF, текст #000000, акцент #C97C5D
3. **Тёмная** — фон #1A1815, карточки #252320, текст #E8E4DB, акцент #C9A66B

## 6. Логотип
- Основной: слово «LOVII» + сердце с двумя листьями
- Цвета: сердце — #C97C5D, листья — #8F9E8B
- Для premium-меток: #C9A66B
- Высота в шапке: 32px

## 7. Визуальные принципы
- Никакого чистого белого фона — только #E9E4D4 или #F0EDDE
- Фотографии: реальные, без стоков, тёплый свет, натуральные текстуры
- Анимации: только микроанимации (плавное наведение, появление при скролле)
- Градиенты: только мягкие, терракота → охра, opacity ≤ 10%

## 8. Сетка и отступы
- Базовая единица: 8px (8, 16, 24, 32, 48, 64, 80)
- Сетка: 12 колонок (desktop), 4 (tablet), 1 (mobile)
- Макс. ширина контента: 1440px

## 9. Ограничения
- Не использовать: чёрный текст, синие/фиолетовые акценты, QR-коды в дизайне, глобусы, Wi-Fi иконки
- Не использовать: резкие тени, неон, sci-fi стили
- Все изображения — без наложенного текста

## 10. Примеры секций для генерации
- Hero: “Цифровая витрина для вашего бизнеса”, фон #E9E4D4, фото фермера с телефоном
- Как это работает: 4 карточки с иконками (регистрация → каталог → заказ → выплата)
- Для кого: микробизнес, франчайзи, партнёры (Фотострана, Яндекс, Igoods)
- Преимущества: “Нулевая комиссия до 30 000 ₽”, “Вывод на карту за 5 минут”
