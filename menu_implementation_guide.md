# Руководство по реализации меню для сайта

## Общая архитектура

Для унификации и повторного использования меню на всех страницах сайта созданы шаблоны в директории `/templates/`:
- `header.html` - шапка сайта с основным меню
- `footer.html` - подвал сайта
- `mobile-nav.html` - мобильное меню, которое выезжает сбоку

## Меню для десктопной версии (в хедере)

### Структура
- Логотип слева
- Основное меню с горизонтальным расположением пунктов
- Блок действий пользователя (поиск, вход, регистрация, переключатель темы) справа

### Особенности
- Пункты меню имеют подчеркивание при наведении
- Адаптивный дизайн для разных размеров экранов
- Пункты меню не длинные, без вложенности для лучшего восприятия

### Стили
- Используются CSS переменные для темизации
- Плавные переходы и анимации
- Современные теневые эффекты

## Мобильное меню (в хедере)

### Структура
- На мобильных устройствах основное меню в хедере скрывается
- Вместо этого используется отдельное мобильное меню, расположенное в нижней части экрана

## Мобильное меню (выезжающее сбоку)

### Особенности
- Кнопка-бургер находится в правом нижнем углу экрана
- При нажатии меню выезжает с правой стороны экрана
- Не интегрировано в хедер, как просил заказчик
- Имеет вложенную структуру с раскрывающимися подразделами

### Структура
- Заголовок с логотипом и кнопкой закрытия
- Основные пункты меню
- Подменю, которое раскрывается при нажатии на основные пункты
- Разделители и специальные элементы (вход/регистрация)

## Интеграция в страницы

### Вариант 1: Использование включений на сервере
```html
<!--#include file="templates/header.html" -->
<!-- Содержимое страницы -->
<!--#include file="templates/footer.html" -->
```

### Вариант 2: JavaScript-инъекция
```javascript
fetch('templates/header.html')
  .then(response => response.text())
  .then(data => {
    document.getElementById('header-container').innerHTML = data;
  });
```

### Вариант 3: Использование шаблонизаторов
В зависимости от используемого фреймворка (PHP, Node.js, Python и т.д.)

## Технологический стек

- HTML5
- CSS3 (с использованием flexbox и grid)
- JavaScript (без использования фреймворков для легкости интеграции)
- Используются современные CSS переменные для темизации
- Иконки Font Awesome

## Адаптивность

- Десктоп (1024px+): Полноценное меню в хедере
- Планшет (768px-1024px): Адаптированное меню
- Мобильные устройства (<768px): Выезжающее боковое меню в нижней части экрана

## Требования к поддержке

- Современные браузеры (Chrome, Firefox, Safari, Edge)
- Поддержка мобильных устройств
- Поддержка темной/светлой темы
- Адаптивный дизайн
